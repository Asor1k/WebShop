Installation guide

This is the nodejs server API. To launch the server you need to install nodejs.

Libraries that need to be installed using npm:

  1. sqlite3
  2. express

Example: 

npm install sqlite

To launch the server navigate to the folder where you saved the server.js file and launch it via cmd command 

node server.js

Retrive the full data

Retrives all phones data that are stored in the data base

GET /getAllItems

Endpoint URL

http:/localhost:3000/getAllItems

Example requests

  curl http:/localhost:3000/getAllItems

Example responses

Response is the array of objects. Each object has:

Name Type Description
id integer Unique identifier of the phone in database
brand string The brand name of the phone
model string The model of the phone
os string The os of the phone
image string Direct link to the image of the phone
screensize integer The screensize of the phone

MIME format: text/json

{
  "data":[{
    "id": 0,
    "brand":"Samsung",
    "model":"Samsung S21",
    "os": "Android",
    "image":"https://cdn1.technopark.ru/technopark/photos_resized/product/1000_1000/250610/1_250610.jpg",
    "screensize":6
  },
  {
    "id": 1,
    "brand":"Apple",
    "model":"Iphone 12",
    "os": "IOS",
    "image":"https://jabko.ua/image/catalog/products/2021/09/142254/2021-09-14%2021.47.41.jpg",
    "screensize":7
  }]
}
Status code Description
200 OK - The request has succeeded. The client can read the result of the request in the data object of response
400 Bad Request - The request could not be understood by the server due to malformed syntax.

Retrive the phone data by phone id

Retrives the phone data using its id in database

GET /getPhone/:id

Endpoint URL

http:/localhost:3000/getPhone/:id

Example requests

  curl http:/localhost:3000/getPhone/2

Query parameters:

Name Type Description
id integer Index used to indicate what phone to get

Add new phone

Adds new item(phone data) to the database

POST /addNewItem

Endpoint URL

http:/localhost:3000/addNewItem

Example requests

  curl http://localhost:3000/addNewItem?brand=Apple&model=Iphone12&os=IOS&image=https://mobiel-product.imgix.net/q384zbe1fthns9irqpryj2qlwxum&screensize=6

Query parameters:

Name Type Description
brand string The brand name of the phone
model string The model of the phone
os string The os of the phone
image string Direct link to the image of the phone
screensize integer The screensize of the phone
Status code Description
201 Created - The request has been fulfilled and resulted in a new phone data added
400 Bad Request - The request could not be understood by the server due to malformed syntax.

Update phone data

Updates phone data using the phone id in the data base.

PUT /updatePhone/:id

Endpoint URL

http:/localhost:3000//updatePhone/:id

Example requests

  curl http://localhost:3000/updatePhone/1?brand=Apple&model=Iphone12&os=IOS&image=https://mobiel-product.imgix.net/q384zbe1fthns9irqpryj2qlwxum&screensize=6

Query parameters:

Name Type Description
id integer Unique identifier of the phone in database
brand string The brand name of the phone
model string The model of the phone
os string The os of the phone
image string Direct link to the image of the phone
screensize integer The screensize of the phone
Status code Description
200 OK - The request has succeeded. The phone data was updated.
400 Bad Request - The request could not be understood by the server due to malformed syntax.

Delete phone data

Delete phone from database using the given id

DELETE /deletePhone/:id

Endpoint URL

http:/localhost:3000/deletePhone/:id

Example requests

  curl http://localhost:3000/deletePhone/1

Query parameters:

Name Type Description
id integer Unique identifier of the phone in database
Status code Description
200 OK - The request has succeeded. The phone data was deleted from database.
400 Bad Request - The request could not be understood by the server due to malformed syntax.
